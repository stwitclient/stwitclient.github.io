Inspired By Metro.

Drawing inspiration from Microsoft's Metro design language, sTwit celebrates an elegant and clean user interface that's beautiful to look at and a pleasure to read.

Made for .

Taking advantage of the latest Microsoft .NET and Windows Presentation Foundation frameworks, sTwit is a first-class application that shines on Windows.
